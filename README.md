# Dr. AI - Healthcare Chatbot 

An AI-powered medical assistant built with **Flask** and **Google Gemini AI**. This project helps users identify potential diseases based on symptoms and suggests over-the-counter medicines.

##  Key Features
- **Hybrid Diagnosis:** Uses a local dataset for common symptoms and Gemini AI for complex queries.
- **Aesthetic UI:** Modern Glassmorphism design with floating medical symbols and animations.
- **Responsive:** Works seamlessly across different devices.

##  Tech Stack
- **Backend:** Python, Flask
- **AI Model:** Google Gemini 1.5 Flash
- **Frontend:** HTML5, CSS3 (Advanced Animations), JavaScript (Fetch API)
